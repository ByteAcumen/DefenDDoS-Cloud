Problem Statement & Goal

Background: As organizations move to the cloud, their web applications become prime targets for Distributed Denial-of-Service (DDoS) attacks, which threaten service availability.

The Challenge: The primary difficulty is distinguishing a malicious DDoS attack from a legitimate traffic spike, as both can cause similar performance issues. An intelligent and automated real-time defense system is needed.

Project Goal: To develop a sophisticated analytics tool that accurately identifies various DDoS attack patterns in real-time—such as volumetric floods, uniform behavioral profiles, and unusual traffic timing—to effectively counter the threat and ensure service continuity.

Core Objectives

  • Detect threats using AI and Machine Learning.
  • Visualize network status on a real-time dashboard.
  • Automate threat mitigation with AWS WAF integration.
  • Create a tamper-proof audit log using Blockchain.
  • Build a scalable and resilient microservices architecture.

Innovation & Novel Contributions

  • Hybrid ML Model: Combines Isolation Forest for speed with LSTM for deep analysis, resulting in superior detection.
  • Blockchain Audit Trail: Creates a tamper-proof, immutable log of all security events, guaranteeing data integrity.
  • End-to-End Automation: Implements a complete, closed-loop from threat detection to immediate, programmatic mitigation in the cloud.
